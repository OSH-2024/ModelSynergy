# LLM Agent

### 简介

**本项目的重点在于如何将LLM与操作系统耦合，使得其在内核与用户间提供一个更用户友好的、更便于操作的中间层。因此，我们将LLM的上下文机制引入操作系统，使得操作系统的文件系统等成为上下文的组成部分，提高LLM对用户的个性化适用性**
